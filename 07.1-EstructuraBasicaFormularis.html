Estructura bàsica d'un formulari

<>L'estructura bàsica d'un formulari es compon de 4 elements HTML que tenen aquestes etiquetes.
  • <form>
  • <label>
  • <input>
  • <button>

ETIQUETES

<form>

L'elemnt HTML form té una etiqueta d'obertura <form> i una de tancament </form>. Les etiquetes form engloben tot el formulari. Accepta atributs globals.

<label>

L'element HTML label té una etiqueta label que serveix per escriure el nom del camp a emplenar. Té etiqueta d'onertura i tancament.

<input>

L'element input és el que serveix perquè l'usuari pugui introduir una informació. Depenent de l'atribut que porti es veurà d'una manera o d'una altra. L'atribut per defecte és un camp de tiups text en una sola línia. Hi ha 22 tipus de valors en l'atribut type.

<button>

L'element HTML button crea un botó que permet enviar el formulari. Té etiqueta d'obertura i tancament.
