Select

Element HTML select bàsic

L'etiqueta select serveix per seleccionar entre una llista d'opcions un o més ítems.

Si es posa l'etiqueta select sola, activa una llista deseplegable però no es veu res.

Etiqueta option

Cada etiqueta option afegeix un ítem per poder triar en el menú desplegable. El select porta l'atribut name i les etiquetes option els atributs value.

Atribut múltiple

L'atribut múltiple, que es posa dins l'etiqueta select, ens permet escollir-ne més d'un si apretem la tecla Control
