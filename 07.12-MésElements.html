MÉS ELEMENTS DELS FORMULARIS

Etiqueta fieldset

S'utilitza per agrupar elements dins d'un formulari. Té etiqueta d'obertura i de tancament. Sovint va acompanyada per l'etiqueta legend.

Input tipus file

Serveix per afegir un fitxer al formulari.

Dades personals

Etiqueta meter

A part de l'atribut indispensable value, porta atributs min, max, low, high i optimum.

Etiqueta progress

Porta els atributs value i max. Crea una barra de progress en funció dels atributs value i max. Una bona pràctica és treballar amb percentatges. Dues barres, una amb max 200 i l'altra amb max 100 tenen la mateixa longutud.

Etiqueta textarea

Porta els atributs cols i rows amb valors numèrics
