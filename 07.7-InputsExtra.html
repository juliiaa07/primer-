INPUTS EXTRA

Mirarem un grup d'inputs que no tenen una classificació fàcil

Input tipus color

Un input tipus color ens permet triar un color en una paleta de colors. Si es fa una pàgina on l'usuari hagi d'escollir un color per algun item en concret o le color de fons d'una pàgina.

Input tipus number

Un input de tipus number només ens deixa posar números. Les petites fletxes que surten dins el camp d'entrada de dades per pujar i baixar el número es diuen widgets. Aquest input s'ha d'utilitzar per temes numèrica que no siguin dates.

Input tipus range

Un input de tipus range ens pinta una barra per seleccionar un valor. Accepta l'atribut step que determina de quant en quant es mou la nostra barra. També té l'atribut min i max

Input tipus reset

Es crea un botó que restableix tots elss camps del fomulari
