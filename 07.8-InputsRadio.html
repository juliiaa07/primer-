Input de tipus radio

Un input de tipus radio ens permet construir caselles de selecció múltiple.

Atributs importants

Atribut name

Hem d'associar les opcions amb l'atribut name amb el mateix valor a totes les opcions.

Atribut value

Necessitem donar un valor a cada opció perquè el servidor s'enregistri

Atributs accessoris booleans

Hi ha alguns atributs booleans que poden anar bé en certes ocasions.

  • checked
  • disabled
  • autofocus
  • required

Atribut checked

Et selecciona una opció

Atribut disabled

Prohibeix una opció

Atribut autofocus

Suggereix una opció

Atribut required

Obliga a escollir una opció
